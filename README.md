# Usage
This is a motif for [Mugen EVE HD 1.1](https://mugenarchive.com/forums/downloads.php?do=file&id=18053-complete-eve-battle-classic-hd-dj-van-doom-guy-) that supports 1080p and has 2,172 character slots.

# Setup
Put these files in the "data" folder, which is located in the base directory for Mugen. Right click mugen.exe > Create Shortcut, then right click the shortcut > Properties > set Target to "mugen.exe -r night", and run it.
